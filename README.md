📝 DESCRIPCIÓN DEL PROYECTO
CONCIERTOS CONECTADOS ES UNA APLICACIÓN WEB DE ARQUITECTURA "SINGLE PAGE APPLICATION" (SPA) DISEÑADA PARA LA GESTIÓN, VENTA Y RESERVA DE ENTRADAS A CONCIERTOS Y EVENTOS MUSICALES EN COLOMBIA. LA PLATAFORMA PERMITE A LOS USUARIOS EXPLORAR EVENTOS POR CIUDAD O CATEGORÍA, DETALLAR LA INFORMACIÓN DE CADA UNO, GESTIONAR UN CARRITO DE COMPRAS EN TIEMPO REAL Y REGISTRAR SUS DATOS DE FACTURACIÓN. ADEMÁS, CUENTA CON UN PANEL DE ADMINISTRACIÓN PROTEGIDO QUE PERMITE DE FORMA INTEGRAL EL CONTROL DE INVENTARIO Y EL HISTORIAL DE VENTAS.

🚀 CARACTERÍSTICAS PRINCIPALES
VISTA DE CLIENTE (HOME):

BÚSQUEDA EN TIEMPO REAL: FILTRADO AUTOMÁTICO DE EVENTOS POR NOMBRE A MEDIDA QUE SE ESCRIBE.

FILTROS AVANZADOS: SEGMENTACIÓN DETALLADA POR CIUDADES (BARRANQUILLA, BOGOTÁ, BUCARAMANGA, MEDELLÍN, CÚCUTA) Y CATEGORÍAS MUSICALES (ROCK, POP, REGGAETON, VALLENATO).

DISEÑO ADAPTATIVO: INTERFAZ COMPLETAMENTE OPTIMIZADA PARA DISPOSITIVOS MÓVILES Y PANTALLAS DE ESCRITORIO MEDIANTE MEDIA QUERIES PERSONALIZADOS.

SISTEMA DE CARRITO DE COMPRAS:

MÓDULO INTERACTIVO EN FORMATO MODAL PARA EL CONTEO, VISUALIZACIÓN DINÁMICA Y CÁLCULO AUTOMÁTICO DE LOS TOTALES ACUMULADOS.

FORMULARIO COMPLETO PARA RECOLECCIÓN DE DATOS DE FACTURACIÓN (IDENTIFICACIÓN, NOMBRE, DIRECCIÓN, TELÉFONO Y CORREO ELECTRÓNICO).

PANEL DE ADMINISTRACIÓN CENTRALIZADO:

CONTROL DE ACCESO SEGURO: LOGIN RESTRINGIDO PARA EL ADMINISTRADOR (CORREO: admin@mail.com | CLAVE: 123456).

MÓDULO DE CATEGORÍAS: ADICIÓN Y ELIMINACIÓN DE GÉNEROS MUSICALES.

MÓDULO DE EVENTOS: CREACIÓN DE NUEVOS EVENTOS (CÓDIGO, NOMBRE, CIUDAD, PRECIO, FECHA, HORA, IMAGEN Y DESCRIPCIÓN) Y BAJAS EN EL SISTEMA.

MÓDULO DE VENTAS: HISTORIAL COMPLETO DE TRANSACCIONES CON ORDEN CRONOLÓGICO Y CAPACIDAD DE DESPLIEGUE DETALLADO POR CLIENTE.

PERSISTENCIA DE DATOS LOCAL:

SISTEMA BASADO EN LOCALSTORAGE PARA GUARDAR EN MEMORIA DEL NAVEGADOR EL ESTADO DE LAS COMPRAS, CATEGORÍAS ACTUALIZADAS Y EVENTOS. INCLUYE UNA CARGA DE DATOS SEMILLA DE FORMA AUTOMÁTICA EN LA PRIMERA APERTURA.
